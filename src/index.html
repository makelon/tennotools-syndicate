Tracking

Select the syndicate level from a dropdown menu by clicking on the rank name, then enter the reputation points for that level in the upper input field. The bottom input field is used to calculate how a given amount of added standing for one syndicate will affect the others. Both inputs can be either positive or negative and support expressions containing addition and subtraction.

Use the Update button to set the new ranks and reputation as the starting point, or the Save button to store the results locally if a supported web browser is used.
